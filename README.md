# 🏦 Belong Bank - Secure CLI System
A personalized banking experience powered by Python.
Belong Bank is a Python-based Command Line Interface (CLI) application that reimagines traditional banking systems. Built by a 1st-year B.Tech student, this project bridges the gap between robust backend security and human-centric user experience.

### 🛡️ Security Feature
This application uses the `hashlib` library to implement **SHA-256 one-way hashing** for secure PIN storage.
* **Input:** `1234`
* **Stored as:** `03ac674216f3e15c761ee1a5e255f067953623c8b388b4459e13f978d7c846f4`

Even if the database file is accessed, user credentials remain mathematically protected.

### ✨ Key Highlights
* **Emotional Intelligence:** Personalized time-based greetings and empathetic error handling.
* **Data Persistence:** Real-time ledger management using **Pandas** and Excel.
* **Robust Validation:** Strict checks for PIN length and positive transaction amounts.
